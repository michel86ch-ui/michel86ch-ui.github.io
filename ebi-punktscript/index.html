    BACnet .PresentValue (Wert/schreiben), .PresentValue/ORDN (binär lesen)
    C-Bus (Excel5000) .PointValue (analog), .PointState/ORDN (binär lesen), .PointState (binär schreiben), .ModeState (Hand=1/Auto, lesen via .ModeState/ORDN)

    Das System wird pro Punkt explizit angewählt – das Property-Dropdown zeigt dann nur die dort zulässigen Properties. Punktnamen sind auf jeder Anlage anders benannt, darum gibt es bewusst keine automatische Erkennung. Über «eigene…» lassen sich trotzdem beliebige Properties eintragen.

    Beim Schreiben auf einen C-Bus-Punkt zuerst ModeState = 1 setzen (Hand), sonst wird der Wert vom Controller wieder überschrieben – der «ModeState-Guard» in der Wertübergabe macht genau das.
